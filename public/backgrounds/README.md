# רקעי אמנות להזמנות

כאן מניחים תמונות רקע שהכיתוב של ההזמנה משתלב עליהן — כמו איורי הכותל,
פרחים או עיטורים בהזמנות מעוצבות. **הזיהוי אוטומטי**: קובץ בתיקייה = אפשרות
רקע בעורך, בלי שום חיבור.

## מבנה התיקיות

- `backgrounds/shared/` — רקעים שמוצעים ב**כל** התבניות (למשל עיטור כללי).
- `backgrounds/<slug>/` — רקעים לתבנית מסוימת בלבד, למשל:
  - `backgrounds/wedding/כותל בצבעי מים.png` — יופיע רק בהזמנה לחתונה
  - `backgrounds/bar-mitzvah/תפילין.png` — רק בבר מצווה

שם הקובץ (בלי הסיומת) הוא השם שמוצג בעורך — אפשר ורצוי בעברית.

## הנחיות לתמונה

- **פורמט**: PNG / JPG / WebP / SVG.
- **יחס**: כיחס העמוד — A5 עומד (למשל ‎1748×2480 ל-300dpi). התמונה נמתחת
  ל-cover וממורכזת.
- **קריאות הכיתוב**: עדיף מרכז בהיר או מעומעם (כמו בהזמנות מקצועיות —
  האיור בשוליים והמרכז נקי לטקסט).
- כשנבחר רקע תמונה, מסגרת הזהב האוטומטית מוסתרת — האמנות היא המסגרת.

הקובץ `shared/עיטור-זהב.svg` הוא דוגמה מובנית — אפשר למחוק או להחליף.
